# Simulator_Pojezdu_Gravirky
Zápočtový projekt VCP - Zadání 7) Simulátor pojezdu gravírky

Program simuluje pohyb hlavy laseru na pracovní ploše v rovině. Udržuje informaci o
aktuální a výchozí (nastavitelné) poloze, umožňuje zadat pohyb buď o určitý vektor, nebo na
konkrétní souřadnici, a vypočítává celkovou uraženou vzdálenost. Má také určité možnost
ukládání a načítání sekvencí pohybů. Konečně obsahuje různé ochrany (například před
vstupy, které vedou mimo pracovní oblast).

● reprezentace polohy a výpočty vzdálenosti
● pohyb relativní i absolutní
● ukládání a načítání definovaných ras
